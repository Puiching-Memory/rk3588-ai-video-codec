Detailed Description

RGA 硬件缩放/格式转换 API。

基于 Rockchip RGA 2D 加速器，支持 NV12/YUV420P 等格式的 硬件缩放、裁剪和格式转换。零 CPU 占用。

对帧进行硬件缩放。

使用 RGA 2D 加速器完成缩放，零 CPU 占用。 输入输出均为 rkvc_frame*，内部自动管理缓冲区。

Parameters
src源帧 (NV12/YUV420P)
out输出帧 (caller 负责 unref)
cfg缩放配置
Returns
RKVC_OK 成功, RKVC_ERR_HW RGA 不可用, 其他错误码
rkvc_frame *scaled = NULL;
rkvc_scale_config cfg = { .dst_width = 1280, .dst_height = 720 };
rkvc_frame_scale(src_frame, &scaled, &cfg);
// 使用 scaled ...
rkvc_frame_unref(scaled);
