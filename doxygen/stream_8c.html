Detailed Description

实时流式处理 API 实现。

内置环形缓冲区，支持异步 push/pull，适合生产者-消费者模式。 编码流: 原始帧 push → 内部编码 → 编码包 pull 解码流: 压缩包 push → 内部解码 → 解码帧 pull
